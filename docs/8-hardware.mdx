# Hardware
Depai is compatible with a wide range of hardware configurations on Mac, Windows, and Linux platforms.

## Minimum System Requirements

- Memory: 8GB
- Disk Space: 40GB
- Network Bandwidth: 10MB/s

## Suggested System Requirements

- Memory: 64GB
- Disk Space: 500GB
- Network Bandwidth: 500MB/s

**Note**: Depai currently supports two models: Gemma2:27B and Llama3:8B. If your GPU has 24GB or more of VRAM, we recommend running Gemma2:27B, as it will earn you 3 times the points compared to Llama3:8B.
